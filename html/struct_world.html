Detailed Description

Essentially a set of application-wide global variables defining the dimensions of the "world" and the conversion factors from pixel to world units and back, as well as a few rendering constants.

inlinestatic

Function called through the initialization of a global variable in the main program. Although the user specifies dimensions for the rendering pane, the function may set different values that agree better with the world aspect ratio.

Parameters
xminMinimum x value to get mapped to the display area.
xmaxMaximum x value to get mapped to the display area.
yminMinimum y value to get mapped to the display area.
ymaxMaximum y value to get mapped to the display area.
paneWidthuser-set width of the redering pane
paneHeightuser-set height of the redering pane

If the two ratios differ by more than 5%, then reject the dimensions
